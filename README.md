🚀 **Antigravity YouTube Deep Explorer**

YouTube Data API v3를 활용한 고성능 다크모드 유튜브 메타데이터 탐색 도구

Antigravity YouTube Deep Explorer는 일반적인 유튜브 검색을 넘어 영상의 태그, 정확한 수치, 채널의 상세 정보까지 한눈에 파악할 수 있는 프리미엄 웹 도구입니다. 설치가 필요 없는 단일 HTML 기반으로 설계되어 어디서나 즉시 사용할 수 있습니다.

✨ **주요 기능 (Key Features)**

🔍 Deep Search: 영상 제목뿐만 아니라 태그(Keywords), 좋아요 수, 상세 설명 등 모든 메타데이터를 추출합니다.

📊 Channel Insights: 해당 영상 채널의 구독자 수와 프로필 정보를 실시간으로 조회합니다.

🎨 Cyber-Dark UI: 네온 포인트와 다크 모드가 적용된 미래지향적이고 직관적인 디자인을 제공합니다.

🎬 Premium Animation: GSAP를 활용한 부드러운 스태거(Stagger) 등장 효과 및 인터랙션을 경험할 수 있습니다.

🔒 Secure API Management: 사용자의 API 키를 서버가 아닌 브라우저 내 localStorage에 안전하게 저장합니다.

📱 Fully Responsive: 데스크톱, 태블릿, 모바일 등 모든 기기에서 완벽한 레이아웃을 지원합니다.

🛠 **사용 기술 (Tech Stack)**

Frontend: HTML5, JavaScript (ES6+)

Styling: Tailwind CSS (JIT CDN)

Animation: GSAP (ScrollTrigger, Stagger)

Icons: Lucide Icons

API: YouTube Data API v3

🚀 **시작하기 (Getting Started)**

1. 전제 조건
이 프로그램을 사용하려면 YouTube Data API v3 키가 필요합니다.

Google Cloud Console에서 프로젝트를 생성하고 API 키를 발급받으세요.

2. 실행 방법
이 저장소를 클론하거나 index.html 파일을 다운로드합니다.

index.html 파일을 브라우저(Chrome, Edge 등)로 엽니다.

우측 상단의 설정(Settings) 아이콘을 눌러 본인의 API 키를 입력합니다.

검색어를 입력하고 유튜브의 깊은 데이터를 탐색하세요!
